"editor.type" can be set with the folowing values:
  • color
  • date
  • datetime
  • datetime-local
  • email
  • month
  • number
  • range
  • search
  • tel
  • time
  • url
  • week
To more details, see http://www.w3schools.com/html/html_form_input_types.asp
